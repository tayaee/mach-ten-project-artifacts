# Zen Garden Match Three

A relaxing match-three puzzle game with calm vibes and engaging gameplay.

## Description

A classic 3-match puzzle game with peaceful background aesthetics. Players swap adjacent flowers on an 8x8 grid to create matches of 3 or more identical flowers in a row or column. The game features a 60-second time limit, combo scoring, and bonus items for matching 4+ flowers.

## Rationale

Uses familiar match-three mechanics for low barrier to entry, optimized for office workers and students seeking quick stress relief.

## Details

- 8x8 grid with 5 types of flower icons
- Swap adjacent flowers to match 3+ horizontally or vertically
- Matched flowers disappear; new flowers fall from above
- 60-second time limit to maximize score
- Match 4+ flowers for bonus items that clear surrounding cells

## How to Build

No build required. Pure HTML5, CSS3, and Vanilla JavaScript in a single file.

## How to Run

Open `index.html` in any web browser (Chrome, Edge, Safari, Firefox).

## Examples

Game ends with final score and maximum combo count displayed on screen, providing a sense of achievement.
