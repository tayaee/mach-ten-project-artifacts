# Neon Snake Retro

Navigate a snake to eat food while avoiding walls and your own body in this classic arcade game.

## Description

A grid-based snake game where you control a growing snake. Eat food to score points and grow longer. The game speeds up as you score higher, increasing the challenge. Avoid hitting walls or your own tail.

## Features

- 20x20 grid gameplay
- Snake starts at length 3 in center
- Arrow keys or WASD controls
- Speed increases with each food eaten
- Score tracking with local storage high score
- Neon visual style

## How to Build

No build process required. HTML5 Canvas with pure JavaScript.

## How to Run

Open `index.html` in any modern web browser.

## Controls

- Arrow keys or WASD: Change direction
- Cannot directly reverse direction (180-degree turns blocked)

## Examples

Start the game and use arrow keys to guide the snake toward the pink food. Each food eaten increases score by 10, grows the snake by one segment, and slightly increases speed. The game ends when the snake hits a wall or its own body.
