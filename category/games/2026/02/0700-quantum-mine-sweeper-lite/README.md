# Quantum Mine Sweeper Lite

Classic minesweeper with the added fun of logical deduction.

## Description

A logic puzzle game based on the classic minesweeper rules that everyone knows. High accessibility due to familiar rules, and clear logical structure making it ideal for AI agent coding.

## Details

Find 15 mines on a 10x10 grid. When you click a tile, the number of adjacent mines is displayed. Clicking an empty tile with no numbers automatically opens all adjacent empty tiles. Right-click to place flags on suspected mine locations. Win by opening all safe tiles (excluding mines), lose immediately when clicking a mine.

## How to Build

Single `index.html` file using HTML5, CSS3, and JavaScript. No build process required - runs directly in web browsers.

## How to Run

Open the generated `index.html` file in a web browser (Chrome, Edge, etc.) to play.

## Examples

Players analyze number combinations to find safe paths and feel achievement when finding the last mine and opening all tiles. Beginners start with 10x10, but difficulty can be adjusted through code modification.
