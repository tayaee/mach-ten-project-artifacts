# Sliding Number Classic Puzzle

A classic 15-puzzle game where you slide numbered tiles to arrange them in order.

## Description

A traditional sliding tile puzzle with a 4x4 grid containing numbered tiles from 1 to 15 and one empty space. Move tiles adjacent to the empty space to arrange all numbers in sequential order.

## Features

- 4x4 grid with numbers 1-15
- Click tiles adjacent to empty space to move them
- Move counter to track your moves
- Timer to track your solving time
- Success message when puzzle is solved
- New Game button to reset and shuffle

## How to Run

Open `index.html` in a web browser (Chrome, Edge, Safari, Firefox).

## Examples

Click on any tile adjacent to the empty space to slide it. For example, if tile 7 is next to the empty space, clicking 7 moves it into the empty position. Continue sliding tiles strategically until all numbers are arranged 1-15 in order, with the empty space in the bottom-right corner.

## Tech Stack

- HTML5
- CSS3 (CSS Grid for layout)
- Vanilla JavaScript (ES6+ classes)
